# 🤖 Модератор-бот для Telegram

Бот для модерации групп Telegram. Администраторы могут выдавать предупреждения, мутить, банить и кикать участников. При достижении лимита предупреждений (по умолчанию 5) бот автоматически выдаёт мут.

## 📋 Возможности

- Выдача предупреждений (`/warn`)
- Просмотр количества предупреждений (`/warns`)
- Мут на определённое время (`/mute`)
- Снятие мута (`/unmute`)
- Бан (на время или навсегда) (`/ban`)
- Разбан (`/unban`)
- Кик (`/kick`)
- Статус пользователя (предупреждения, мут, бан) (`/status`)
- Команды доступны администраторам чата и владельцу бота (указан в `config.py`)

## 🛠 Установка и запуск

### 1. Клонируйте репозиторий или скачайте файлы

### 2. Установите зависимости
bash
pip install aiogram
Другие библиотеки не требуются — всё остальное встроено в Python.

### 3. Получите токен бота у @BotFather
Создайте нового бота командой /newbot

Скопируйте полученный API-токен.

### 4. Настройте конфигурацию
Отредактируйте файл config.py:

python

API_TOKEN = "ВАШ_ТОКЕН"        # замените на реальный токен

OWNER_ID = 123456789           # ваш Telegram ID (число)

MUTE_MINUTES = 10              # время мута по умолчанию

WARN_LIMIT = 5                 # предупреждений до мута

DB_NAME = "moderator.db"       # файл базы данных

### 5. Запустите бота
bash
python bot.py
### 6. Добавьте бота в группу
Сделайте бота администратором группы (обязательно дайте право «Блокировка пользователей»).

Группа должна быть супергруппой. Если это обычная группа, преобразуйте её через настройки чата.

##📖 Команды
/start	Показать приветствие и список команд.

/warn [@user или ID]	Выдать предупреждение пользователю. Если предупреждений станет 5 – автоматический мут.

/warns [@user или ID]	Показать количество предупреждений у пользователя.

/mute [@user или ID] [минуты]	Замутить пользователя на указанное количество минут (по умолчанию 10).

/unmute [@user или ID]	Снять мут.

/ban [@user или ID] [минуты]	Забанить пользователя. Без указания минут – навсегда.

/unban [@user или ID]	Разбанить пользователя.

/kick [@user или ID]	Кикнуть пользователя из чата.

/status [@user или ID]	Показать статус (предупреждения, мут, бан).

Как указывать пользователя

Можно ответить на сообщение пользователя командой (например, /warn в ответ на его сообщение) или указать числовой ID (например, /mute 123456789 15). Указание по @username не поддерживается для упрощения логики.

## 📁 Структура файлов
bot.py – главный файл с обработчиками команд.

logic.py – все функции для работы с базой данных (SQLite).

config.py – настройки (токен, OWNER_ID, константы).

moderator.db – база данных (создаётся автоматически при первом запуске).

## 🗄 База данных
Используется SQLite. Таблицы:

users – информация о пользователях (телеграм ID, имя, текущие предупреждения, статус мута/бана).

warn_history – история выданных предупреждений.

moderation_actions – логи всех действий администраторов (варны, муты, баны и т.д.).

##⚙️ Примечания
Бот использует только команды.

Для работы команд /mute, /ban, /kick бот должен быть администратором супергруппы.

Владелец бота (указанный в OWNER_ID) может выполнять любые действия даже без прав администратора в чате.

После мута/бана пользователь физически не может писать в чат (используется restrict_chat_member и ban_chat_member).

## 🐛 Возможные ошибки и их решение
Ошибка	Решение
bot is not a member of the group chat	Бот не добавлен в группу или был удалён. Добавьте бота заново и сделайте администратором.
method is available for supergroup and channel chats only	Группа не преобразована в супергруппу. Сделайте это в настройках чата.
Not enough rights	У бота нет необходимых прав администратора (особенно «Блокировка пользователей»).
NameError: name 'timedelta' is not defined	Обновите код: в bot.py должен быть импорт from datetime import datetime, timedelta.
Предупреждения всегда 0/5	Пользователь отсутствовал в базе. Исправлено в последней версии – пользователь создаётся автоматически.
